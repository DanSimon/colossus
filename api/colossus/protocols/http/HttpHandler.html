colossus.protocols.http

HttpHandler

class HttpHandler extends ServiceServer[HttpRequest, HttpResponse] with ColossusNavigator

An instance of ConnectionHandler mixed with ColossusNavigator. This class is responsible for taking the decoded HttpRequest and examining it's method & path to figure out which helper to delegate to.
